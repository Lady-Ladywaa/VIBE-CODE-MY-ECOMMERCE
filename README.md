# 👗 Wa-Vibe Fashion

## 📖 Project Overview

| รายละเอียด | ข้อมูล |
|------------|---------|
| 🏪 **ชื่อร้าน** | Wa-Vibe Fashion |
| 👗 **ประเภทธุรกิจ** | ร้านจำหน่ายและให้เช่าเสื้อผ้าแฟชั่นมุสลิม |
| 🛍️ **สินค้า** | ชุดเดรส, ผ้าคลุมฮิญาบ, เครื่องประดับ (Accessories), กระเป๋า |
| 💼 **รูปแบบการขาย** | ซื้อขาด (Sale) และเช่า (Rental) |
| 👩 **กลุ่มเป้าหมาย** | ผู้หญิงมุสลิม นักเรียน นักศึกษา และวัยทำงาน |

---

# 🎯 Problem Statement

ปัญหาที่พบในปัจจุบัน

| 🚨 ปัญหา | รายละเอียด |
|----------|------------|
| 💰 **ชุดมีราคาสูง** | ชุดมุสลิมสำหรับออกงาน เช่น ชุดรายอ หรือชุดไปงานแต่ง มีราคาค่อนข้างสูง เนื่องจากเป็นงานตัดเย็บที่ประณีต ทำให้หลายคนไม่สามารถซื้อได้ |
| 👗 **ซื้อมาใส่เพียงครั้งเดียว** | ลูกค้าส่วนใหญ่มักใช้ชุดเพียง 1-2 ครั้ง ทำให้ไม่คุ้มค่ากับเงินที่จ่าย |
| 📅 **ไม่มีระบบเช่าที่เป็นมาตรฐาน** | Marketplace ทั่วไป เช่น Facebook หรือ Shopee ไม่รองรับระบบเช่าที่ครบถ้วน เช่น การเลือกวันเช่า การตรวจสอบคิว และการคืนเงินมัดจำ |

---

# 💡 Proposed Solution

พัฒนาเว็บไซต์ **Hybrid E-Commerce**

รองรับทั้ง

- 🛒 การซื้อสินค้า (Sale)
- 👗 การเช่าสินค้า (Rental)

พร้อมระบบจัดการหลังบ้าน เช่น

- 📅 ปฏิทินการเช่า
- 💰 คำนวณเงินมัดจำอัตโนมัติ
- 📦 ตรวจสอบสถานะสินค้า
- 🔄 จัดการการคืนสินค้า
- 💳 คืนเงินมัดจำหลังตรวจสอบสินค้า

---

# ⭐ จุดแข็ง (Strengths)

- 👗 รองรับทั้ง **การซื้อและการเช่า** ในเว็บไซต์เดียว
- 💎 มีบริการเสริม เช่น แต่งหน้า คลุมฮิญาบ และเช่าเครื่องประดับ
- 💰 ช่วยลูกค้าประหยัดค่าใช้จ่าย โดยไม่จำเป็นต้องซื้อชุดราคาแพง
- 📅 มีระบบจองวันเช่า ลดปัญหาการเช่าซ้ำ
- 🧕 เหมาะกับลูกค้ากลุ่มมุสลิมโดยเฉพาะ

---

# ⚠️ จุดอ่อน (Weaknesses)

- 📦 การจัดการสต็อกมีความซับซ้อน เพราะมีทั้งสินค้าขายและสินค้าเช่า
- 🛠️ หากลูกค้าคืนสินค้าชำรุด อาจส่งผลต่อการให้บริการลูกค้ารายถัดไป
- 📅 ระบบเช่าต้องมีการจัดการวันเช่าและวันคืนอย่างแม่นยำ
- 💰 ต้องมีระบบตรวจสอบก่อนคืนเงินมัดจำ
